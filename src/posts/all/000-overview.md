# 前言

大家好，我是老马。

最近在追《凡人修仙传》，很是上头，也很是感慨。

修仙一途，大道无情，前路漫漫。

而你我所在的人间，又何尝不是如此呢？

为什么会写这个系列呢？

一直想系统的提升自身的综合能力，多年以来，苦于不得法门，也没有看到过比较符合自己的书籍。

于是花了大量的时间，梳理了一套知识体系，希望可以帮助到有同样困惑的小伙伴。

后续可以抽时间，分门别类简单介绍一下，方便大家有所了解。
